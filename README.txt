Квест-мастер: Генератор приключений  
Практическая работа №8 — Программирование на Python  

Полноценное GUI-приложение на **PyQt6**, полностью соответствующее техническому заданию:

- Генератор квестов с валидацией, автосохранением и горячей клавишей Ctrl+Enter  
- Редактор карт 800×600 (фон #f4e4bc, кисть, маркеры, текст Uncial Antiqua, ластик, сохранение PNG/JPG)  
- Экспорт пергаментов в PDF (weasyprint) и DOCX (python-docx) по трём шаблонам Jinja2   
- SQLite-базa с системой версионности (таблица `quest_versions`)  
- Геймификация: уровни, прогресс-бар, достижения, звук при получении XP  
- Босс-файт: генерация 100 квестов меньше чем за 5 сек → +20 XP (тест проходит за ~0.03 сек)
